
How to Install
==============

Requirements
-----------
- python >= 3.11
- numpy
- scipy
- pandas
- mne
- tslearn
- autograd
- pymanopt[autograd]
  
You can install all the dependencies with the following pip command.

>>> pip install numpy scipy pandas mne tslearn autograd pymanopt[autograd]

Install with pip
----------------
>>> git clone https://github.com/simonkojima/nearby.git
>>> cd ./nearby
>>> pip install .
